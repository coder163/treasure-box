1. IO流

1.1. 概述

对数据的一切读写都是以流的形式实现,Java的读写都是以Java虚拟机为参照物的,所以读是in写就是out

一般情况下IO流如果想使用必须要和File类进行绑定

所有的流在使用完成之后都需要关流,注意关闭顺序正好是开启的反向顺序

1.2. 分类

按照流行分：out、in

按照类型划分

  • 字符流：以记事本的方法打开文件,能够看的懂的内容,一般使用字符流
  • 字节流：以记事本的方法打开文件,能够看不懂的内容,一般使用字节流

注意：因为计算机中所有的数据都是以字节的形式存储,所以字符流能做的事情字节流都可以做!

1.3. 体系结构

字符流

  • java.io.Reader 是字符输入流的顶级类

  • java.io.Writer 字符输出流的顶级类

字节流

  • java.io.InputStream 字节输入流的顶级类
  • java.io.OutputStream 字节输出流的定义类

转换流：当我们以字节流读取字符时使用的一些缓冲流
