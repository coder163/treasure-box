Java版本的划分

JavaSE:java开发的标准版本(基础版本),就是我们现在学习的

JavaEE：Java的企业级开发版本,J2EE其实指的是一个语言(技术)体系,中高级学习的

JavaME：当初SUN公司为移动设备开发准备的一个版本,现在不怎么流行

Java中关于包的概念

作用：包机制是为了解决在大型项目中,Java文件冲突的文件

常用快捷键

行注释/取消行注释:Ctrl+?

段落注释：Ctrl+shift+?

取消段落注释：Ctrl+shift+|

代码提示键：ALT+?

复制一行：Ctrl+alt+方向键下

移动一行：ALT+上下键

删除一行:Ctrl+D

自动导包：Ctrl+shift+O

自动格式化代码：Ctrl+shift+f 默认情况下跟搜狗输入法冲突
